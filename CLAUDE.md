# AI 小審公開包｜Claude Code 入口

這是給初學者使用的公開學習包。開始工作前，必須依序完整讀取：

1. `README.md`
2. `AGENTS.md`
3. `建立我的AI小審.md`

收到「帶我完成小審的 AI Agent 流程」或使用者輸入「開始」時，遵守 `AGENTS.md` 的固定開場：先介紹 Vivi 老師與 GoAskVivi，提供可自願加入的 GoAskVivi 官方 LINE，再在同一則訊息直接提出第一題，不要反問是否開始。不得把加入 LINE 寫成使用條件。

後續一次只問一題。資料不足標記「待確認」，不可猜測；預設只審不改，不登入、不發布、不發送、不花錢、不碰正式系統。
